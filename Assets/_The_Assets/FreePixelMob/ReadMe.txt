FreePixelMob
art: benmhenry@gmail.com
code: davidahenry@gmail.com

Description:
Includes 16 by 16 animated (UI.Image & Sprite) slime monster.<br>
Includes walk, attack and three idle animations.<br>
The animation controller will select a random idle animation each time.

Documentation:
Sprite:
1. Create->2D->Sprite
2. Add Animator and select slime animation

UI.Image:
1. Create->UI->Image
2. Add Animator and select slime animation

Files:
FreePixelMob/Mob.cs : example script switches between animation states
FreePixelMob/ReadMe.txt : this
FreePixelMob/Slime.controller : animation controller with walk, attack, and idle animations
FreePixelMob/SlimeSprite.unity : example scene
FreePixelMob/SlimeImage.unity : example scene
FreePixelMob/SlimeA.png : all sprites on single sprite sheet
FreePixelMob/StateRandom.cs : used in slime animation controller to select random idle animation
